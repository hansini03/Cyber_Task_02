# Threat Analysis

## 1. Insider Threat

### Description
A threat originating from employees or authorized users.

### Possible Impact
Data theft, information leakage, and unauthorized activities.

### Prevention Methods
- Access control
- Monitoring user activities
- Security awareness programs

---

## 2. Malware

### Description
Malicious software designed to damage or disrupt systems.

### Possible Impact
Data loss, financial damage, and system compromise.

### Prevention Methods
- Antivirus software
- Regular updates
- Safe browsing practices

---

## 3. Phishing

### Description
Fraudulent attempts to steal sensitive information through emails or fake websites.

### Possible Impact
Credential theft and financial loss.

### Prevention Methods
- Employee awareness training
- Email filtering
- Multi-Factor Authentication

---

## 4. Weak Passwords

### Description
Passwords that are easy to guess or crack.

### Possible Impact
Unauthorized access to accounts and systems.

### Prevention Methods
- Strong password policies
- Password managers
- MFA implementation

---

## 5. Unpatched Systems

### Description
Systems that have not received security updates.

### Possible Impact
Exploitation of known vulnerabilities.

### Prevention Methods
- Regular patch management
- Vulnerability assessments
- Automated updates
